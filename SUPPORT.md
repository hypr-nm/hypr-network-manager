# Support Policy

## Scope

Support covers:

- Installation and build issues
- Runtime/network management regressions
- Configuration and theming questions

Support does not guarantee:

- Distribution-specific packaging maintenance
- Custom compositor integration beyond documented behavior
- Immediate response for feature requests

## How To Get Help

1. Search existing issues first.
2. Open a new issue with:
   - Environment (distro, compositor, GTK/NetworkManager versions)
   - Steps to reproduce
   - Expected vs actual behavior
   - Relevant logs
3. Use clear titles and minimal reproducible steps.

## Lifecycle

- Active support: current release line
- Best-effort support: previous minor line until next release
- Unsupported: older versions
